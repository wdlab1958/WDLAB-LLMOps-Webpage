왜 WDLAB-LLMOps 인가?

Cloud SaaS LLMOps의 종속성을 끊고, 데이터 주권 · 규제 준수 · 멀티 에이전트 가시성을 단일 자체 호스팅 플랫폼으로 통합합니다.

⌘

완전한 데이터 주권

모든 트레이스 · 평가 · 감사 로그가 온프레미스에 저장. 외부 SaaS 전송 0건. ALLOW_EXTERNAL_LLM=false가 기본값.

⚖

AI 기본법 자동 준수

제31조 ① ② (고지·공개), 제34조 ① ② ④ (영향평가·결정근거·이의제기) 증거를 자동 누적, 월간 컴플라이언스 리포트(MD/DOCX/PDF) 생성.

⊞

멀티 에이전트 네이티브

LangGraph · CrewAI · AutoGen 등 39개 프레임워크를 Maestro 5계층 DAG로 표준화 추적.

◈

Zero-Cost 로컬 추론

Ollama · vLLM 기반 gemma3 / qwen3 / llama3.2 로컬 모델로 운영. 외부 LLM 호출 자동 차단(prefix block).

▣

WORM + Merkle 무결성

컴플라이언스 감사 · 가드레일 · 온톨로지 이벤트를 Append-only + HMAC-SHA256 체인으로 봉인. 변조 시 즉시 탐지.

◐

XAI 5계층 설명

Raw → Feature → Natural Language → Visual → Actionable. /explain API로 규제 기관에 결정 근거 제공.

Workspace · Observability

관측의 5가지 시야

📊

Dashboards

Cost · Latency · Errors · Feedback Score 시계열. 샘플 분포 히트맵, 자동 새로고침 토글.

📋

Logs · Traces

실시간 트레이스 타임라인 테이블. 프로젝트 · 모델 · 프레임워크 필터, 상세로 딥링크.

🔬

Trace Detail

Gantt 타임라인 + Span 트리 + 토큰 어텐션 히트맵. /explain XAI 5계층 응답.

💬

Threads View

thread_id 기준 대화 버블. 누적 비용/지연/평가점수, 사용자 피드백 인라인.

🔗

DAG Flow

Maestro Dispatch → Worker Agents → Merge. WebSocket 라이브 업데이트로 실행 중 그래프가 자라는 모습 관찰.
